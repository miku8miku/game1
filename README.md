## 功能特性

- **3D太空战斗场景**：使用Three.js引擎打造沉浸式3D太空环境
- **高级视觉效果**：集成Bloom辉光、粒子系统、屏幕震动等特效
- **霓虹美学设计**：赛博朋克风格的色彩搭配和UI设计
- **流畅操作体验**：鼠标控制飞船移动与射击
- **智能敌人AI**：敌机自动追踪并攻击玩家
- **实时战斗反馈**：爆炸效果、碰撞检测和得分系统
- **响应式UI界面**：包含血量条、得分显示等游戏信息
- **复古CRT特效**：模拟老式显示器扫描线效果
- **动态光影系统**：实时光照和材质反射效果
- **流畅动画系统**：包含飞船倾斜、后坐力等细节动画

## 快速开始

1. 下载项目文件
2. 直接在支持WebGL的现代浏览器中打开 `117.html` 文件
3. 点击"INITIATE"按钮开始游戏
4. 使用鼠标控制飞船移动和射击

## 解读规则

- **控制方式**：移动鼠标控制飞船位置，按住鼠标左键进行射击
- **目标**：击毁迎面而来的敌方飞行器以获取分数
- **生命值系统**：初始生命值为100%，每次与敌机碰撞损失20%生命值
- **得分机制**：击毁每个敌机获得100分
- **游戏结束条件**：当生命值降至0%时游戏结束
- **视觉反馈**：击中敌人或受到伤害时会有屏幕震动效果
- **飞船行为**：飞船会根据移动方向进行倾斜模拟真实飞行感
- **敌人行为**：敌机会从远处向玩家方向移动，速度随机

## 技术栈

- **HTML5**：页面结构和Canvas容器
- **CSS3**：样式设计，包括霓虹效果、动画和响应式布局
- **JavaScript (ES6 Modules)**：游戏逻辑和交互控制
- **Three.js**：3D图形渲染引擎
- **WebGL**：硬件加速的3D图形渲染
- **后期处理系统**：
  - EffectComposer：后期处理合成器
  - RenderPass：基础渲染通道
  - UnrealBloomPass：Bloom辉光效果
- **字体**：Google Fonts Orbitron字体用于赛博朋克风格文本
- **响应式设计**：适配不同屏幕尺寸的窗口调整处理
